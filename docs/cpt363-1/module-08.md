# Chapter 8 新型雷达与数据融合

本章面向现代船桥和智能航海场景。学生需要理解固态雷达、相控阵、多波束、雷达图像处理和多源数据融合如何扩展传统雷达的应用。

## 学习目标

- 比较磁控管雷达、固态雷达、相控阵雷达和多波束雷达。
- 了解雷达图像增强、目标检测和深度学习处理的基本用途。
- 分析雷达、AIS、ECDIS、视觉和气象数据融合的价值。
- 说明雷达数据在 VTS、远程操控和自主船舶中的作用边界。

## 核心问题

1. 固态雷达相对传统雷达有哪些运行优势？
2. 雷达图像智能识别能解决哪些问题，又会带来哪些新风险？
3. 多源融合怎样提高态势感知，为什么也需要一致性检查？
4. 智能航海场景中，雷达数据的价值如何被重新定义？

## 教学主线

1. 从传统雷达限制进入新型雷达技术。
2. 介绍雷达图像处理和智能识别。
3. 讲解雷达与 AIS、ECDIS、视觉数据的融合逻辑。
4. 连接 VTS、远程操控、MASS 和未来航海人才能力。

## 新技术对照

| 技术方向 | 课程解释 | 风险边界 |
| --- | --- | --- |
| 固态雷达 | 提高可靠性，支持灵活波形和较强信号处理能力 | 仍需理解显示、杂波和目标观测限制 |
| 相控阵 / 多波束 | 改善扫描方式和空间覆盖 | 成本、集成和解释复杂度上升 |
| 图像增强 | 改善弱目标和复杂场景判读 | 算法可能放大或隐藏特征 |
| 目标识别 | 支持智能航海态势感知 | 训练数据和海况变化影响可靠性 |
| 多源融合 | 连接雷达、AIS、ECDIS、视觉和气象数据 | 一致性检查和责任边界仍需人工判断 |

## 课堂训练

1. 让学生比较磁控管雷达和固态雷达在启动、维护、探测和信号处理方面的差异。
2. 给出一个智能航海系统场景，让学生说明雷达、AIS、视觉和 ECDIS 分别提供什么信息。
3. 让学生分析“算法识别为船舶，但雷达回波不稳定”时应如何复核。

## 复习问题

- 多源融合提高态势感知时，为什么仍需检查数据一致性？
- 智能航海场景中，雷达数据的核心价值是什么？
