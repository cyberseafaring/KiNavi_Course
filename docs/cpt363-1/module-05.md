# Chapter 5 雷达导航与定位方法

本章进入雷达的航行应用。学生需要使用雷达距离、方位、固定目标、岸线和平行线导航完成船位判断和受限水域监控。

## 学习目标

- 使用距离、方位和距离方位联合方法进行雷达定位。
- 理解固定目标选择、岸线识别和定位误差来源。
- 掌握平行线导航、通过距离监控和转向点检查。
- 说明雷达叠加到电子海图时需要检查哪些偏差。

## 核心问题

1. 什么样的目标适合作为雷达定位参考？
2. 雷达距离和方位误差会怎样影响船位判断？
3. 平行线导航如何帮助受限水域航行？
4. 雷达图像与 ECDIS 不一致时应该先检查什么？

## 教学主线

1. 从固定目标和岸线回波进入雷达定位。
2. 对比距离定位、方位定位和联合定位。
3. 讲解平行线导航的设置、监控和偏离判断。
4. 用港区、狭水道和近岸场景组织应用练习。

## 导航应用拆解

| 应用 | 适用场景 | 检查点 |
| --- | --- | --- |
| 距离定位 | 目标明显、距离读数可靠时 | 目标前沿、量程、海图标注 |
| 方位定位 | 方位稳定、目标清晰时 | 艏线误差、航向输入、目标识别 |
| 距离方位定位 | 港区、沿岸、狭水道 | 目标是否固定、是否与海图一致 |
| 平行线导航 | 受限水域、通过距离监控 | 安全距离、海图基准、显示模式 |
| 雷达叠加复核 | Radar overlay / ECDIS 对照 | 位置源、航向源、图层偏移 |

## 课堂训练

1. 给出一段狭水道航线，让学生选择三个适合雷达定位的固定目标。
2. 让学生画出一条平行线，并说明通过距离和偏离判断。
3. 给出雷达岸线与 ECDIS 图层不一致的情形，让学生列出检查顺序。
