# Chapter 1 雷达技术基础与船舶应用

本章建立船舶导航雷达的基本概念。学习重点是理解雷达如何完成探测、测距和方位判断，以及为什么在 GNSS、AIS、ECDIS 普及之后，雷达仍然是驾驶台安全判断的核心设备。

## 学习目标

- 解释 Radar / Radio Detection and Ranging 的含义和基本工作过程。
- 说明船舶导航雷达在目标探测、航行定位、避碰和海况观察中的作用。
- 初步理解 SOLAS、IMO 性能标准和船舶配备要求。
- 能用一个雾航或夜航场景说明雷达信息的安全价值。

## 核心问题

1. 雷达如何通过发射电磁波和接收回波发现目标？
2. 雷达能提供哪些不同于 GNSS、AIS 和视觉瞭望的信息？
3. 船舶导航雷达的核心功能怎样服务航行安全？
4. 设备性能标准为什么会影响实际操作和课程学习？

## 教学主线

1. 从日常回声类比进入雷达测距原理。
2. 介绍雷达在船舶导航中的历史发展和现实作用。
3. 对比目标探测、岸线识别、避碰支持和海况观察。
4. 引入船载雷达性能要求，为后续系统组成和操作章节铺垫。

## 关键概念拆解

- **回波 / Echo**：目标反射回来的电磁波信号，是雷达图像的基础。
- **距离 / Range**：从本船到目标的距离，常用海里表示，是避碰和定位的基础数据。
- **方位 / Bearing**：目标相对本船或真北的方向，受艏线和航向输入影响。
- **显示模式 / Display mode**：Head-Up、North-Up、Course-Up 等模式服务不同操船任务。

## 课堂训练

1. 给出一个雾航场景，让学生列出视觉、AIS、ECDIS 和雷达分别能提供的信息。
2. 让学生解释“为什么 AIS 上没有目标，雷达上仍可能存在目标”。
3. 用一张简单雷达截图，让学生标出距离、方位、岸线、目标和可能的杂波区域。
