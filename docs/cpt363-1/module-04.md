# Chapter 4 雷达操作与图像判读

本章把雷达原理和系统组成转化为驾驶台操作流程。学生需要掌握雷达开机检查、参数调节、显示模式和目标判读的基本顺序。

## 学习目标

- 按步骤说明雷达开机、待机、发射、调节和关机过程。
- 合理选择量程、增益、调谐、海杂波和雨杂波设置。
- 使用 EBL、VRM、光标读数和显示模式支持目标判断。
- 判读船舶、岸线、浮标、雨区、干扰和遮挡等常见图像。

## 核心问题

1. 雷达图像调节为什么需要固定顺序？
2. 量程变化会怎样改变目标密度和风险感知？
3. 海杂波抑制过强会带来什么危险？
4. Head-Up、North-Up、Course-Up 显示分别适合哪些操作？

## 教学主线

1. 建立标准化雷达操作流程。
2. 用图像变化解释主要旋钮和菜单设置。
3. 结合 EBL / VRM 讲解距离和方位测量。
4. 通过典型图像训练目标类型和异常现象识别。

## 操作检查单

| 步骤 | 目标 | 风险 |
| --- | --- | --- |
| 安全确认 | 确认天线附近无人员、无危险作业 | 忽视发射安全 |
| 选择量程 | 匹配远距离态势、会遇分析或近距离操船 | 量程不当导致目标密度或细节误判 |
| 调节增益 | 让背景噪声和目标回波保持可辨 | 过低漏检，过高淹没目标 |
| 调节海雨杂波 | 改善近区或雨区图像 | 过度抑制弱小目标 |
| 使用 EBL / VRM | 形成可记录的距离和方位数据 | 忽略显示模式和误差来源 |

## 课堂训练

1. 给出一张“过高增益”的雷达图像描述，让学生写出调节顺序。
2. 让学生解释 Head-Up、North-Up 和 Course-Up 在靠泊、沿岸和海图对照中的差异。
3. 设计一个“雨区中小目标观察”操作流程，要求写出每次调节后的复核动作。

- EBL / VRM 读数进入避碰判断前应检查哪些条件？
