## Chapter-to-Practice Mapping

- 第1章 -> 用一个雾航场景说明雷达的不可替代作用。
- 第2章 -> 绘制船载雷达系统组成图，标出外部传感器输入。
- 第3章 -> 分析一张包含海杂波、雨区或岸线的回波图像。
- 第4章 -> 给出一套雷达图像调节顺序和检查理由。
- 第5章 -> 用固定目标完成距离方位定位或平行线导航设计。
- 第6章 -> 对一个会遇目标完成 CPA / TCPA 和避碰行动分析。
- 第7章 -> 根据故障现象写出可能原因和检查步骤。
- 第8章 -> 说明雷达、AIS、ECDIS、视觉融合的价值和风险。
