# Chapter 2 船载雷达系统组成

本章从设备组成和数据输入角度理解船载雷达。学生需要把天线、发射机、接收机、显示单元、控制单元和外部传感器连接成一条完整的信息链。

## 学习目标

- 识别船载雷达系统的主要硬件组成。
- 说明发射、接收、信号处理和显示之间的流程。
- 理解罗经、计程仪、GNSS、AIS、ECDIS 和 VDR 对雷达显示的影响。
- 能画出一张船载雷达系统组成图。

## 核心问题

1. 雷达天线、发射机和接收机分别承担什么任务？
2. 航向、航速和位置输入为什么会影响目标矢量？
3. AIS 和 ECDIS 接入雷达后，驾驶台工作流程有什么变化？
4. 哪些设备异常会直接改变雷达图像或目标计算结果？

## 教学主线

1. 从雷达信号流进入硬件组成。
2. 比较磁控管雷达和固态雷达的基本差异。
3. 讲解显示单元、控制面板和用户设置。
4. 连接外部传感器输入，说明综合导航系统中的雷达角色。
