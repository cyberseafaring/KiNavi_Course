# Chapter 2 船载雷达系统组成

本章从设备组成和数据输入角度理解船载雷达。学生需要把天线、发射机、接收机、显示单元、控制单元和外部传感器连接成一条完整的信息链。

## 学习目标

- 识别船载雷达系统的主要硬件组成。
- 说明发射、接收、信号处理和显示之间的流程。
- 理解罗经、计程仪、GNSS、AIS、ECDIS 和 VDR 对雷达显示的影响。
- 能画出一张船载雷达系统组成图。

## 核心问题

1. 雷达天线、发射机和接收机分别承担什么任务？
2. 航向、航速和位置输入为什么会影响目标矢量？
3. AIS 和 ECDIS 接入雷达后，驾驶台工作流程有什么变化？
4. 哪些设备异常会直接改变雷达图像或目标计算结果？

## 教学主线

1. 从雷达信号流进入硬件组成。
2. 比较磁控管雷达和固态雷达的基本差异。
3. 讲解显示单元、控制面板和用户设置。
4. 连接外部传感器输入，说明综合导航系统中的雷达角色。

## 系统链路

| 环节 | 课堂解释 | 失效或误差影响 |
| --- | --- | --- |
| 天线系统 | 决定波束方向、扫描范围和方位分辨力 | 遮挡、旁瓣、盲区、方位误差 |
| 发射机 | 产生雷达脉冲或固态发射信号 | 探测距离下降、无回波、调谐异常 |
| 接收机 | 放大和选择微弱回波 | 噪声增大、弱小目标漏失 |
| 显示处理 | 形成 PPI 图像、测量读数和目标信息 | 图像失真、矢量错误、告警不可靠 |
| 外部输入 | 提供航向、航速、位置、AIS 和海图信息 | 真运动、叠加和 CPA / TCPA 偏差 |

## 课堂训练

1. 让学生画出“目标回波从天线到屏幕”的信号流程。
2. 给出“真矢量方向异常”的故障现象，让学生判断可能是雷达回波问题还是航向 / 航速输入问题。
3. 对比磁控管雷达和固态雷达，让学生列出维护、启动和探测特性上的差异。

## 复习问题

- 为什么现代雷达需要理解为“回波设备 + 传感器输入 + 显示处理”的综合系统？
- 航向输入错误会影响哪些雷达功能？
- AIS 和 ECDIS 接入雷达后，哪些判断更方便，哪些风险更隐蔽？
