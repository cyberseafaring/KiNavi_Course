# Chapter 3 电磁波传播与目标探测

本章解释雷达图像背后的物理机制。学生需要理解电磁波传播、目标反射、雷达截面积、环境干扰和探测限制如何共同决定雷达屏幕上的回波。

## 学习目标

- 说明雷达波的传播、反射、折射、散射和衰减。
- 理解目标尺寸、材料、形状、姿态和距离对回波强弱的影响。
- 分析海杂波、雨杂波、同频干扰、旁瓣和异常传播。
- 能解释为什么某些目标会漏检、弱显或形成虚假回波。

## 核心问题

1. 同一艘船在不同距离和角度下为什么回波不同？
2. 海杂波和雨杂波分别怎样影响近距离和大范围探测？
3. 雷达截面积为什么比目标真实尺寸更能说明可探测性？
4. 雷达图像中的亮点是否总是代表真实目标？

## 教学主线

1. 从波长、频率、脉冲宽度和波束宽度进入探测性能。
2. 讲解海上典型目标的反射特性。
3. 分析杂波、遮挡和干扰对图像的影响。
4. 把物理限制转化为后续图像调节和风险判断。

## 目标探测要点

| 影响因素 | 现象 | 操作提示 |
| --- | --- | --- |
| 目标尺寸和高度 | 大型高目标回波强，小型低目标容易弱显 | 近距离和恶劣海况下保持谨慎 |
| 目标材料和形状 | 钢质目标通常反射强，木质或低矮目标反射弱 | 用连续观测确认目标稳定性 |
| 海况和雨区 | 海杂波、雨杂波可能遮蔽真实目标 | 杂波抑制要逐步调整 |
| 岸线和建筑 | 强回波、阴影区和多次反射并存 | 对照海图和量程变化复核 |
| 雷达安装 | 桅杆、烟囱、甲板货物可能形成盲区 | 了解本船雷达阴影扇区 |

## 课堂训练

1. 给出三种目标：大型商船、小型渔船、浮标，让学生比较雷达回波特点。
2. 给出海杂波和雨杂波场景，让学生写出调节和复核步骤。
3. 让学生用“目标、环境、设备、操作”四个维度解释一次漏检风险。

## 复习问题

- 雷达截面积为什么不等同于目标真实面积？
- 海杂波抑制为什么可能带来新的危险？
- 判断一个亮点是否为真实目标时，至少需要哪些复核证据？
