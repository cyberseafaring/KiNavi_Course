# Chapter 7 维护、测试与故障诊断

本章建立雷达设备可靠运行的检查和诊断思路。学生需要把故障现象、可能原因、检查步骤和航行安全管理连接起来。

## 学习目标

- 说明雷达日常检查、开航前检查和运行中监控要点。
- 识别距离误差、方位误差、调谐异常和显示异常。
- 根据故障现象提出可能原因和排查顺序。
- 理解维护记录、缺陷报告和航行限制的管理意义。

## 核心问题

1. 雷达性能测试应该覆盖哪些项目？
2. 方位误差、距离误差和图像异常分别可能来自哪里？
3. 天线、波导、电源、接收机和显示单元故障有什么典型表现？
4. 设备缺陷如何转化为航行限制和风险控制措施？

## 教学主线

1. 从日常检查清单进入性能测试。
2. 建立故障现象和可能原因之间的映射。
3. 分析维护记录、标准要求和船舶安全管理。
4. 用案例训练诊断步骤和报告表达。

## 故障分析框架

| 故障现象 | 可能来源 | 航行影响 |
| --- | --- | --- |
| 无回波或回波弱 | 发射机、接收机、天线、调谐或增益设置 | 探测能力下降 |
| 方位整体偏差 | 艏线校准、天线编码、航向输入 | 定位和避碰判断偏差 |
| 距离读数异常 | 时基、显示处理、量程设置或测距校准 | 船位和通过距离错误 |
| 目标拖尾或图像异常 | 显示处理、增益、杂波控制或干扰 | 图像判读困难 |
| ARPA 跟踪不稳定 | 目标弱、海况差、传感器输入异常 | CPA / TCPA 不可靠 |

## 课堂训练

1. 给出一个故障现象，让学生从设备、输入、环境和操作四类原因展开排查。
2. 让学生写出“是否可以继续使用雷达”的条件和航行限制。
3. 让学生模拟交接班，说明需要记录哪些雷达状态和缺陷。

## 复习问题

- 雷达维护怎样同时影响设备可靠性和驾驶台安全判断？
- 方位误差会如何影响平行线导航和避碰标绘？
- 雷达缺陷记录应包含哪些内容，才能支持后续风险控制？
