# Workshop 5 复习与项目展示

本训练用于课程收束。目标是把八章知识重新归并为一条可解释的安全判断链，而不是停留在分散术语记忆。

## 复习主线

1. 雷达如何形成回波。
2. 设备和传感器输入如何影响显示。
3. 操作者如何调节图像并判读目标。
4. 雷达如何服务定位、平行线导航和受限水域监控。
5. 连续目标观测如何转化为避碰判断。
6. 设备故障和多源融合如何改变风险评估。

## 展示题目

学生可从以下题目中选择一个完成 5 分钟展示或 1500 字小论文提纲：

- 雷达在能见度受限航行中的作用与限制。
- 海杂波抑制对近距离小目标探测的影响。
- 平行线导航在狭水道航行中的应用。
- 手工标绘与 ARPA 自动跟踪的比较。
- AIS 与雷达目标不一致时的驾驶台判断。
- 固态雷达和智能识别对传统雷达操作的影响。

## 展示结构

1. 场景：说明船舶所处水域、交通、能见度和设备状态。
2. 信息：列出雷达、AIS、ECDIS、视觉或其他传感器提供的信息。
3. 判断：说明如何从信息得出风险判断。
4. 行动：说明操作、避让、复核或维护建议。
5. 反思：说明该场景中雷达的价值和限制。

## 项目评价量规

| 维度 | 优秀表现 |
| --- | --- |
| 教材连接 | 能明确说明展示内容对应教材章节和课程模块 |
| 场景完整 | 包含水域、交通、能见度、设备状态和关键限制 |
| 数据解释 | 能解释雷达、AIS、ECDIS、视觉或其他信息的来源和可信度 |
| 安全判断 | 能从观测数据推导风险，并说明行动依据 |
| 复核意识 | 能说明行动后、故障后或数据不一致时如何复核 |
| 双语表达 | 中文逻辑清晰，英文术语准确，例如 CPA, TCPA, ARPA, radar overlay, false echo |

## 推荐展示模板

1. **Title / 题目**：一句话说明场景和问题。
2. **Scenario / 场景**：水域、能见度、交通密度、本船状态。
3. **Radar Evidence / 雷达证据**：距离、方位、目标趋势、杂波或图像限制。
4. **Cross-check / 交叉验证**：AIS、ECDIS、视觉、声号或其他来源。
5. **Decision / 判断**：定位、导航、避碰或维护决策。
6. **Reflection / 反思**：雷达的价值、限制和人工复核点。

## 期末复习清单

- 能解释回波形成、距离方位测量和显示模式。
- 能画出雷达系统组成和外部传感器输入。
- 能说明量程、增益、海杂波和雨杂波调节风险。
- 能识别目标、岸线、杂波、假回波和误差来源。
- 能用雷达完成定位、平行线导航和受限水域监控。
- 能用标绘、ARPA、AIS 和 COLREGs 组织避碰判断。
- 能把故障现象转化为检查步骤、航行限制和记录内容。
