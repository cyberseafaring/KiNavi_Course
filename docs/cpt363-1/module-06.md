# Chapter 6 目标跟踪与避碰决策

本章围绕雷达目标运动和会遇风险展开。学生需要理解手工标绘、ARPA、相对运动、真运动、CPA、TCPA、AIS 关联和 COLREGs 之间的关系。

## 学习目标

- 用连续雷达观测判断目标相对运动。
- 理解 CPA、TCPA、BCR、BCT 和目标矢量的含义。
- 比较手工标绘、ARPA 自动跟踪和 AIS 信息。
- 能根据雷达信息分析会遇局面并复核避让行动效果。

## 核心问题

1. 连续两个或三个观测点怎样形成目标运动判断？
2. ARPA 给出的 CPA / TCPA 为什么仍需人工复核？
3. 地面稳定和海面稳定显示对真矢量有什么影响？
4. 雷达目标和 AIS 目标不一致时如何处理？

## 教学主线

1. 从相对运动线进入手工标绘。
2. 讲解 ARPA 获取、跟踪、矢量和告警。
3. 对比雷达目标、AIS 目标和综合显示。
4. 结合 COLREGs 分析会遇、追越、交叉相遇和避让复核。

## 避碰分析流程

| 环节 | 课堂动作 | 产出 |
| --- | --- | --- |
| 连续观测 | 记录目标多个位置、距离和方位 | 相对运动趋势 |
| 手工标绘 | 画相对运动线和最近会遇点 | CPA / TCPA 判断 |
| ARPA 对照 | 查看自动跟踪矢量和告警 | 计算结果复核 |
| AIS 关联 | 检查身份、航向、航速和位置一致性 | 信息可信度判断 |
| 规则分析 | 判断对遇、交叉、追越或特殊水域约束 | 行动责任 |
| 行动复核 | 观察新矢量、新 CPA 和通过距离 | 避让效果 |

## 课堂训练

1. 给出三次目标观测位置，让学生画出相对运动线并估算 CPA / TCPA。
2. 给出 ARPA 和 AIS 不一致的数据，让学生判断哪些信息可用于避碰。
3. 让学生说明避让行动后如何用雷达确认风险已经降低。
