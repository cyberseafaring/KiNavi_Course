# 课程导读 · Course Guide

欢迎来到 `CPT363-1 / 船舶导航雷达`。这是一门双语课程，课堂讲解以中文组织学习路径，同时保留关键英文术语、教材表达和航海设备语境，帮助学生在中文理解和英文专业阅读之间建立连接。

Welcome to `CPT363-1 / Marine Navigation Radar`. This bilingual course uses Chinese to build the learning structure and English to preserve professional radar, bridge, and maritime-navigation terminology.

## 课程定位 · Course Positioning

课程面向航海、交通运输、海事工程、智能航运和海上安全相关方向的本科生。学习者需要掌握的重点是理解雷达信息如何进入真实驾驶台工作：在夜间、雾航、狭水道、港区交通和复杂会遇场景中，船员如何把回波图像转化为航行判断。

The course is designed for students in navigation, transportation, maritime engineering, intelligent shipping, and marine safety. The central question is how radar information becomes bridge judgement in night navigation, restricted visibility, confined waters, port traffic, and collision-avoidance situations.

## 学习路径 · Learning Path

1. 先阅读 [内容架构](cpt363-1/contents.md)，理解八章之间的关系。Start with the chapter structure.
2. 按 [课程安排](cpt363-1/schedule.md) 进入每个章节模块。Follow the 16-unit schedule.
3. 课堂展示时打开 [章节课件](cpt363-1/slides.md)，课后细读 [数字教材](cpt363-1/textbook.md) 和对应译文。Use slides in class and textbook chapters after class.
4. 复习阶段使用 [课程专题](cpt363-1/topics.md) 选取一个真实问题，完成案例分析或小论文。Use the topic map for case analysis or a short paper.

## 学习目标 · Learning Outcomes

完成课程后，学生应能：

1. 解释船舶导航雷达的基本工作原理、主要组成和性能指标。Explain radar principles, system components, and performance terms.
2. 根据航行环境调节量程、增益、海杂波、雨杂波、显示模式和测量工具。Tune range, gain, sea clutter, rain clutter, display modes, EBL, and VRM.
3. 判读船舶、岸线、浮标、雨区、干扰和虚假目标的典型回波特征。Interpret echoes from vessels, coastlines, buoys, rain areas, interference, and false targets.
4. 使用 EBL、VRM、平行线导航、手工标绘和 ARPA 信息支持定位与避碰。Use EBL, VRM, parallel indexing, manual plotting, and ARPA data for position fixing and collision avoidance.
5. 说明雷达、AIS、ECDIS、GNSS、罗经和计程仪之间的数据关系。Describe how radar relates to AIS, ECDIS, GNSS, heading, and speed inputs.
6. 对雷达故障、维护检查和现代雷达技术趋势作出基本分析。Analyse radar faults, maintenance checks, and modern radar technology trends.

## 课堂使用建议 · How to Study

- 课前：浏览本周模块页，确认核心问题和术语。Before class, scan the module page and key terms.
- 课中：跟随课件完成图像调节、目标测量和会遇场景讨论。In class, practise display tuning, target measurement, and encounter analysis.
- 课后：用教材译文补充细节，并用专题页中的问题做一页案例卡片。After class, read the textbook notes and prepare a one-page case card.
- 复习：按“原理 -> 图像 -> 测量 -> 跟踪 -> 规则 -> 决策”的顺序重建知识链。For review, rebuild the chain from principles to picture, measurement, tracking, rules, and decision-making.
