# 训练 5 综合演练报告

本训练用于课程结课。学员需要围绕一个智能航行场景，完成从感知到控制的完整分析。

## 报告结构

1. **场景描述**：水域、能见度、交通、任务、平台状态。
2. **感知输入**：雷达、光电、AIS、电子海图、气象和平台状态数据。
3. **AI 处理**：目标识别、态势理解、风险预测、路径规划或控制建议。
4. **操纵判断**：人员如何解释、采纳、修正或拒绝 AI 输出。
5. **控制执行**：舵、车、航迹、动力或动态定位如何响应。
6. **异常处置**：传感器异常、通信受限、模型输出异常或执行机构故障。
7. **安全复核**：接管条件、降级策略、日志记录和训练反思。

## 推荐题目

- 低能见度条件下 AI 感知辅助避碰演练。
- 受限水域智能航迹跟踪与人工接管。
- 多目标会遇场景中的 AI 风险排序。
- 通信受限条件下远程监督与本船决策。
- 传感器欺骗条件下的多源复核。
- 智能船桥显示设计与值更记录。

## 评价量规

| 维度 | 优秀表现 |
| --- | --- |
| 系统性 | 覆盖数据、感知、决策、控制、复核全链条 |
| 航海性 | 判断符合航海规则、操纵性能和安全裕度 |
| AI 理解 | 能准确说明模型能力和失效条件 |
| 军官能力 | 能体现监督、接管、记录和指挥协同意识 |
| 表达质量 | 中文逻辑清楚，关键英文术语准确 |
