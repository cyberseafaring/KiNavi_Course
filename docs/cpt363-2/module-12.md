# Module 12 课程项目设计

这个扩展模块把教材 8 章汇总成一个可执行的小项目。目标不是做大系统，而是做一个“有数据输入、有处理逻辑、有结果输出、有边界说明”的最小船联网应用。

## Project Directions

- AIS / NMEA 解析与质量检查
- 轨迹回放与异常检测
- 多源数据融合与质量标注
- 安全日志分析与最小告警规则
- 船岸协同信息产品原型

## Minimum Deliverables

- 问题定义
- 输入数据说明
- 处理流程图
- 输出结果截图或表格
- 降级策略或风险边界说明
