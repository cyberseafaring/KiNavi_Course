## How This Course Is Structured

1. `教材章节` 提供概念边界、系统机制和工程约束。
2. `范例课程` 把每章拆成两节 45 分钟课时，便于教学组织。
3. `实验项目 + 数据资源 + 交互测验` 用来巩固每章的核心能力。
4. `扩展教程` 负责把 Python、树莓派、视觉、海事数据处理这些能力接到主线知识上。
