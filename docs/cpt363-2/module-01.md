# Chapter 1 物联网技术基础

本章对应教材第 1 章，是整门课的技术底座。学习重点不是记 IoT 定义，而是理解从“物理量 -> 感知 -> 传输 -> 计算 -> 应用”的完整数据链如何形成，并为后续船联网章节建立共同语言。

## Textbook Mainline

- 物联网概念起源、基本特征与发展节点
- 典型应用领域与系统要素
- 传感器网络、数据采集与误差控制
- 嵌入式系统、物联网操作系统与分层设计
- 云计算、边缘计算与云边协同
- 从通用物联网到船联网的衔接
