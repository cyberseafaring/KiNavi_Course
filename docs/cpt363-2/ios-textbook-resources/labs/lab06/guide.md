# 实验目标

本实验用公开数据集完成一次船舶目标检测的最小流程，目标是让学生理解视觉感知在船联网中的位置，并能对检测结果做质量判断。实验不追求训练出高精度模型，更关注数据、流程与误差来源。

## 环境要求

- Python 3.8+
- 可用 GPU 环境更佳，但不是必需
- 建议完成 扩展教程 中的计算机视觉入门内容

## 数据准备

使用 `../../datasets/ship-images/` 中的示例图片完成推理与可视化。数据量较小，适合课堂演示。

## 实验步骤

## 选择检测方案

优先选择可直接运行的开源检测框架或预训练模型。工程上需要把模型版本、输入尺寸与置信度阈值记录下来，避免结果无法复现。

## 推理与可视化

对样例图片执行推理，输出包含边界框、类别与置信度的结果，并把结果叠加到原图上进行可视化。

## 误差分析

挑选一到两张检测失败或误检明显的图片，分析原因并给出改进方向。改进方向可以是数据层面、阈值层面或场景约束层面。

## 预期结果

- 生成带检测框的结果图片
- 给出至少两条误差原因分析
- 明确记录模型与参数，保证可复现
