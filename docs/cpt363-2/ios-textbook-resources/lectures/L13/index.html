L13

船联网安全目标威胁模型

第7章 · 知识点讲解 · 45分钟

教学目标

本节课从航行风险出发建立安全目标，并把威胁模型作为工程沟通工具。

  • 能够说明保密性、完整性、可用性的含义
  • 能够识别典型资产与攻击面
  • 能够给出一个简化威胁模型并说明假设

内容要点

  • 安全目标与航行风险的对应关系
  • 攻击面分类：身份、链路、设备接口、运维通道
  • 风险分级的直觉理解与优先级

课堂节奏

  • 导入：用一个信息误判案例引出完整性问题
  • 讲解：目标定义与攻击面识别
  • 讨论：如何选择优先防护的风险

资源清单

  • 章节资源：第7章总览
  • 案例材料：安全日志样本
  • 配套实验：安全威胁模拟防护

课后思考题

  1. 举例说明完整性受损如何影响航行决策。
  2. 威胁模型中最常见的假设有哪些，假设错误会带来什么后果。
