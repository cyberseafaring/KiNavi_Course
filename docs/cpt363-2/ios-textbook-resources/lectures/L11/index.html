L11

船联网数据解析存储

第6章 · 实验演示课 · 45分钟

教学目标

本节课通过解析示例报文，把数据治理链条从抽象概念落到可操作的工程步骤。

  • 能够区分数据类型分层与时间语义
  • 能够完成简单 NMEA 语句解析与字段校验
  • 能够理解原始层、清洗层、产品层的边界

内容要点

  • 数据类型分层与存储策略
  • NMEA 语句结构与校验
  • 解析结果的字段命名与版本管理意识

课堂节奏

  • 演示：读取样本日志并解析字段
  • 讲解：时间戳与存储分层
  • 练习：为解析结果设计最小数据字典

资源清单

  • 示例数据：见 数据资源 中 AIS 与 NMEA 样本
  • 实验指引：建议配合 Lab03 与 Lab04

课后思考题

  1. 如果解析出的字段名称发生变更，如何保持历史数据可用。
  2. 为什么保存原始层数据对排错与追溯很重要。
