L04

AIS系统海上信息协同基础

第2章 · 知识点讲解 · 45分钟

教学目标

本节课把 AIS 放回到系统协同语境中，强调它提供了什么信息，缺了什么信息，哪些边界条件会导致误判。

  • 能够描述 AIS 的系统构成和基本工作流程
  • 能够概括 AIS 信息内容与常见报文类型
  • 能够说明 AIS 的局限以及补充思路

内容要点

  • AIS 的产生背景与协同价值
  • 船载设备、岸基站、卫星 AIS 的角色分工
  • 动态信息、静态信息、航次信息的基本字段
  • 局限：覆盖、更新频率、可信度边界与异常情况

课堂节奏

  • 导入：用一段 AIS 轨迹回放引出协同需求
  • 讲解：系统构成与信息内容，强调数据含义
  • 讨论：AIS 失效或异常时的工程应对

资源清单

  • 配套章节：第2章总览
  • 实验指引：Lab03 AIS 报文解析
  • 示例数据：AIS NMEA 样本
  • 章节测验：第2章题库

课后思考题

  1. 为什么 AIS 的“信息有”不等价于“信息可信”。
  2. 如果 AIS 更新频率降低，你会优先采用哪一种补充信息源，原因是什么。
