# Module 13 复习与综合应用

这个扩展模块用于期末复习或课程收束，帮助学生把八章内容重新归并为统一框架，而不是八个孤立主题。

## Review Chain

1. 感知：数据从哪里来，质量如何表达
2. 通信：数据如何传，弱网下如何保底
3. 治理：数据如何解析、存储、融合与共享
4. 安全：系统如何保持可信与可用
5. 应用：这些能力如何落到监管、航行支持、运维和绿色航运

## Final Review Tasks

- 任选一章，总结其输入、处理、输出和约束
- 任选一个实验，说明它支撑教材哪一章的哪条主线
- 用“系统边界、数据链、风险边界、应用价值”四个词复盘整门课
