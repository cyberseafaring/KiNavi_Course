# Module 9 数据解析工作坊

这个扩展模块服务于教材第 2、4、5、6 章，把 AIS、NMEA、轨迹和日志数据真正落到“能读、能查、能解释”的工作流上。

## Focus

- 原始报文与结构化字段的映射
- 校验和、时间戳、质量标注
- 从原始层到清洗层的最小处理流程
- 解析错误、缺测、乱序的排查方法
