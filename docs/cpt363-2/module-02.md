# Chapter 2 船联网技术演变

本章对应教材第 2 章，重点是把通用物联网转入航运场景，明确“船联网到底讨论什么，不讨论什么”。尤其要把 AIS 的基础作用、局限和船联网演进主线区分清楚。

## Textbook Mainline

- 船联网概念边界与视角差异
- 对象范围、协同信息类型与典型场景
- 船端数字化、岸端服务体系与 AIS 基础作用
- e-Navigation 共识与关键里程碑
- 船联网典型形态、航区差异与失效模式
- 智能航海、绿色航运与标准化演进
