# Chapter 8 典型应用发展趋势

本章对应教材第 8 章，把前 7 章的能力落到应用系统：船岸协同航行支持、海事监管、远程运维、绿色航运和智能航海。重点是读懂每个应用的输入、处理、输出与运行边界。

## Textbook Mainline

- 船岸协同航行支持与动态优化
- 海事监管服务与应急取证
- 远程运维和状态监控
- 绿色航运服务、能效指标与排放合规
- 智能航海展望、人机协同与责任边界
- 应用需求表达、标准约束与测试评估
