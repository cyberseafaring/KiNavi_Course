# 《智能航海》

**课程性质**：全校本科生选修课
**课程学时**：32学时
**授课教师**：[张若澜](https://nvc.dlmu.edu.cn/info/1063/2428.htm)

## 课程定位

《智能航海》面向对航海、人工智能、机器人、交通运输和未来产业感兴趣的本科生。课程以 MASS（Maritime Autonomous Surface Ships，海上自主水面船舶）为主线，用一次“南京港到鹿特丹港”的完整航次串联智能船舶、智能港口、船岸通信、环境感知、避碰决策、远程控制和法规标准。

学生不需要先具备航海专业背景。课程重点放在“未来自主船舶如何完成一次真实航行”这个问题上，通过案例、视频、项目官网和课堂讨论，帮助学生理解智能航海的基本概念、关键技术和现实难点。

## 核心问题

一艘未来自主航行船舶从南京出发，经长江、近海、远洋航线、运河或关键海峡，最终抵达鹿特丹或阿姆斯特丹港。它需要连续回答六类问题：

1. **看见什么**：雷达、AIS、摄像机、电子海图和气象海况数据如何形成通航环境认知。
2. **如何通信**：船岸之间如何保持可靠的数据交换和远程监控。
3. **怎样决策**：船舶如何规划航线、避让目标、处理突发情况。
4. **如何控制**：推进、舵机、动力和定位系统如何执行航行决策。
5. **谁来负责**：船员、远程操作员、港口、VTS和监管机构如何协同。
6. **如何验证安全**：自主系统如何经过测试、认证、运行限制和应急预案检验。

## 教学特色

- **航次驱动**：用“南京到鹿特丹”的完整航行过程组织课程内容。
- **问题驱动**：每周围绕一个真实技术挑战展开，让技术名词回到具体航行场景。
- **案例驱动**：结合 Yara Birkeland、MEGURI2040、Mayflower Autonomous Ship、Rotterdam Digital Port 等项目。
- **演示驱动**：课堂中保留可点击链接，展示视频、企业官网、项目页面和监管动态。
- **基础友好**：强调概念理解、案例判断和技术兴趣，不要求学生掌握复杂数学推导。

## 学习收获

完成课程后，学生应能：

1. 解释智能航海、MASS、自主航行、远程控制和智能港口的基本概念。
2. 描述一次远洋航次中可能遇到的主要技术挑战。
3. 说明感知、通信、决策、控制、船岸协同和法规标准之间的关系。
4. 对典型自主船舶项目进行初步分析，判断其技术亮点和适用场景。
5. 形成面向未来航运科技的跨学科问题意识。
