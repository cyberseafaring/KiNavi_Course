# 模块 6 船舶控制、操纵响应与自主执行

本模块把 AI 决策落到真实船舶运动。课程重点是船舶运动模型、控制闭环、自动舵、路径跟踪、推进控制、动态定位和低速操纵，使学员理解算法输出如何转化为舵角、车令和航迹变化。

## 学习目标

- 说明船舶操纵中的惯性、舵效、车效、风流浪和响应滞后。
- 理解制导、导航与控制 GNC 的基本关系。
- 认识 PID、模型预测控制、路径跟踪控制和强化学习控制的应用位置。
- 能分析 AI 决策转化为控制执行时的风险和复核方法。

## 核心问题

1. 航线规划结果如何变成舵角、车令和航迹？
2. 船舶控制为什么比车辆或无人机控制更受惯性和环境影响？
3. 自动舵、动态定位和低速操纵控制分别解决什么问题？
4. AI 控制系统如何处理执行机构故障和环境扰动？

## 控制闭环

| 环节 | 内容 | 关键风险 |
| --- | --- | --- |
| 制导 Guidance | 生成期望航迹、航向、速度和转向点 | 航迹不满足水域和任务约束 |
| 导航 Navigation | 估计本船位置、姿态、速度和环境扰动 | GNSS、罗经、计程仪或融合定位异常 |
| 控制 Control | 计算舵角、车令、推进器和侧推指令 | 执行滞后、饱和、故障或过度控制 |
| 反馈 Feedback | 比较计划与实际航迹并修正 | 反馈延迟或传感器误差导致振荡 |

## AI 对操纵的改变

| 操纵任务 | 传统关注点 | AI / 自动控制新增能力 |
| --- | --- | --- |
| 航向保持 | 舵角、偏航、风流修正 | 自适应调参和扰动补偿 |
| 航迹跟踪 | 转向点、偏航量、通过距离 | 基于模型的提前量和误差预测 |
| 低速靠泊 | 舵效车效、侧推、缆绳和风压 | 多推进器协调和仿真训练策略 |
| 动态定位 | 位置保持、推进器分配 | 风流浪预测和故障重构 |
| 编队保持 | 相对方位、距离和航速 | 多船协同控制和通信延迟补偿 |

## 课堂训练

1. 给出一条规划航线，让学员说明控制系统需要哪些状态量。
2. 让学员分析大风横流下自动航迹跟踪可能出现的偏差。
3. 设计一个“AI 控制执行复核表”，包含计划航迹、实际航迹、舵角、车令、误差和接管条件。

## 复习问题

- 智能避碰方案为什么需要考虑船舶操纵性能？
- 自动控制系统出现振荡时，值更官应关注哪些信息？
- 强化学习控制策略进入实船前需要哪些验证？
