## 小论文建议题目

1. 人工智能对舰船值更组织和操纵责任的影响。
2. 多源传感器融合在受限能见度航行中的作用与限制。
3. AI 目标识别系统在复杂水域中的误报漏报风险。
4. 智能避碰系统与 COLREGs 规则解释问题。
5. 强化学习在船舶低速操纵训练中的应用边界。
6. 生成式 AI 在航行辅助决策中的价值和风险控制。
7. 智能船桥显示如何支持军官保持态势主导权。
8. GNSS / AIS 异常条件下智能航海系统的安全降级。
9. 面向海军军官院校的智能航海模拟训练方案。
10. MASS Code 和可信 AI 框架对未来航海教育的启示。

## 展示结构建议

1. **场景**：说明水域、交通、能见度、任务和平台状态。
2. **AI 能力**：说明使用的数据、模型和输出。
3. **操纵影响**：说明它改变了瞭望、判断、决策、控制或复核中的哪一环。
4. **风险边界**：说明数据、模型、通信、执行或人员方面的限制。
5. **军官职责**：说明监督、质疑、接管、记录和复盘要求。
